Don't Let Your Sent Emails End Up In A Spam Folder

SpamSir is a tool that returns a spam score for emails, making it possible to send smarter emails that are not misinterpreted to be spam. In addition to the spam score, SpamSir also returns a report of attributes that alter the score allowing you to decide whether to send certain emails and how to improve your score in the future.

How to Use SpamSir

In order to use SpamSir, you should create a SpamSir account by going to the login screen and selecting the new user option. You don't need an account to receive SpamSir reports however without using an account you do not have access to SpamSir's wide range of built in statistics for your personal emails.

Once you have an account or have decided that you're too cool to use our awesome built in statistic reports, you can start using SpamSir by submitting a post request to spamsir.pontiflex.com/api/submit/ with the parameters: 'email' as the text email (preferably complete with email headers for full functionality), 'username', and 'password'. Upon processing the email you will receive a JSON object complete with spam score and the tests failed. See the full documentation for further detail.
